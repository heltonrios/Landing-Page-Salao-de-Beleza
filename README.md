<h3>🚩 Projeto</h3>
<p>OriginSix é uma página institucional no formato One Page, responsiva, para usar em diversos tipos de micro, pequena e média empresas.</p>
<p>Contém as seguintes seções: 
<li>Header</li> 
<li>Navigation</li>
<li>Home</li>
<li>Sobre</li>
<li>Serviços</li>
<li>Depoimentos</li>
<li>Contato</li>
<li>Footer</li></p>
